# Migracao gradual

## Fase 1

- Criar a estrutura nova.
- Definir padroes de dados.
- Separar imagens por contexto.

## Fase 2

- Copiar apenas as pessoas e paginas confirmadas.
- Validar cada bloco antes de seguir.

## Fase 3

- Reaproveitar o que estiver coerente no projeto antigo.
- Ajustar links, rotas e referencias de imagem.

## Fase 4

- Desativar dependencias antigas quando o novo nucleo estiver coberto.
